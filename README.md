# Ody

내가 쓰는 모든 AI 코딩 도구, 한 곳에서 보고 관리해요.
A local control plane for your AI coding tools — Claude Code, Codex, Gemini, Antigravity.

- **웹사이트 · Website**: https://ody.giml.kr
- **다운로드 · Download**: [Latest release](https://github.com/Mobig9/ody-releases/releases/latest)
- **버그 · 피드백 · Issues**: [New issue](https://github.com/Mobig9/ody-releases/issues/new)

## 이 저장소는요 · About this repo

Ody 앱의 릴리스와 피드백을 위한 공간이에요. 
앱 소스 코드는 비공개로 관리돼요.
This repository hosts releases, release notes and the issue tracker for the Ody desktop app.
The app itself is closed-source.

## 지원 · Support

- macOS (Apple Silicon) — Windows는 준비하고 있어요 · Windows is on the way
- 지원 도구 · Supported tools: Claude Code · Codex · Gemini · Antigravity
