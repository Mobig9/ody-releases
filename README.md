## Ody는요 · What is Ody

Claude Code, Codex, Gemini, Antigravity. 도구가 늘수록 사용량도 설정도 흩어지죠.
Ody는 그 전부를 한 화면에 모아요.

| 기능 | 설명 |
|---|---|
| **사용량 한눈에** | 도구마다 따로 확인하던 사용량을 한 화면에서 봐요. |
| **한도 소진 예측** | 지금 속도면 언제 한도가 끝나는지 미리 알려줘요. |
| **설정 한곳에** | 흩어진 설정 파일을 한 트리에서 편집해요. 바꾼 내역은 되돌릴 수 있어요. |
| **벤더 중립** | 어느 회사 편도 아니라서 전부 보여요. |
| **로컬 퍼스트** | 모든 데이터는 내 컴퓨터에만 있어요. 클라우드에 올라가지 않아요. |
| **트레이 상주** | 평소엔 트레이에 조용히 있어요. 필요할 때만 열어 보면 돼요. |

Ody shows usage across every AI coding tool on one screen and forecasts when a limit
will run out. Scattered config files become one editable tree, with every change
reversible. Vendor-neutral and local-first: nothing leaves your machine. It sits
quietly in your tray until you need it.

자세한 사용법은 [문서](https://ody.giml.kr/docs)에 있어요 · Full manual in the [docs](https://ody.giml.kr/docs).
